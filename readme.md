# MSEFileDialogX a extended MSEFileDialog.

Here how to use it:

Copy in the directory of your project: 

- msefiledialogx.pas
- msefiledialogx.mfm
- msefiledialogx_mfm.pas

Then add at end of uses section:

uses

...

 , msefiledialogx;


This will load the MSEFileDialogX each time a tfiledialog is called.
